Compiling description

1. cd into 'code/fgconverter_2/'
2. type 'make' (this will generate a new "genData" if successful)
3. cd into 'code/clustering/'
4. type 'make' (this will generate a new "learnMoE" if successful)
5. copy 2 executable files (genData and learnMoE) to the upper 'code/init_GMM/' directory
6. now it is ready to run step1 script
